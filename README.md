# Effective Air Quality Prediction Using Machine Learning Algorithms

## 📌 Project Overview
This project focuses on predicting air quality using machine learning techniques. The dataset is preprocessed and multiple models are applied to evaluate and compare prediction performance.

## 📂 Dataset
- **AirQualityUCI.csv**
- Contains environmental data such as CO, NO2, temperature, and humidity.

## ⚙️ Data Preprocessing
The dataset was cleaned and prepared using:
- Handling missing values
- Interpolation techniques
- Median value replacement
- Data normalization (if applied)

## 🤖 Machine Learning Implementation
Each notebook applies similar steps with different models:

- Data cleaning and preprocessing
- Feature selection
- Model training
- Hyperparameter tuning
- Cross-validation
- Learning curve analysis

## 🧠 Models Used
Different machine learning algorithms are implemented across notebooks, such as:
- Linear Regression
- Decision Tree
- Random Forest
- Etc

## 📊 Evaluation Metrics
Models are evaluated using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## 📁 Files in Repository
- `AirQualityUCI.csv` → Dataset
- `.ipynb files` → Implementation of different ML models by group members

## 🚀 How to Run
1. Download the repository
2. Open notebooks in Jupyter Notebook or Google Colab
3. Run cells step by step
